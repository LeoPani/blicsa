# Summary

`Blicsa` (PyBibliomics) is a desktop application written in Python using CustomTkinter that facilitates bibliometric and scientometric analysis. It enables researchers to import, merge, and deduplicate scientific literature records from sources like Scopus, Web of Science, PubMed, OpenAlex, and Crossref.

# Statement of Need

Mapeamento científico and network analysis are central to reviewing literature. Traditional tools (like VOSviewer and Gephi) can be complex for novice users or require pre-processing pipelines. `Blicsa` provides a unified desktop interface for querying APIs directly, clustering literature networks, generating thematic maps, and extracting AI-driven scientific insights.

# Key Features

- **Direct API Search**: Search PubMed, OpenAlex, and Crossref directly from the UI.
- **Network Generation**: Extract co-occurrence, co-authorship, co-citation, and coupling networks.
- **Clustering**: Perform Louvain and Leiden clustering with customizable resolutions.
- **AI Insights**: Generate summaries and label clusters using any OpenAI-compatible API.
